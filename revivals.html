Roblox Revival List

  • RoConomy

    Roconomy is a 2016 Economy simulator like revival that is still in beta. This revival is in beta testing

    Pros:

    • Stable economy
    • Active community

    Cons:

    • In Closed Testing

  • Novetus

    Novetus is a 2007-2012 revival that has been a long-standing part of the revival community and has been there for quite a while - over time it has been updated with new features that solidify its place in this community.

    Pros:

    • Active community
    • Great source for old roblox clients
    • Has some pre-made rbxl's to mess around with

    Cons:

    • You have to self-host to play multiplayer
    • Some clients have vulnerabilities

  • Finobe

    Finobe is an revival that brings retro and fun. Playable on PC. Unfortunately this is not the original Finobe by Raymonf.

    I don't have any knowledge about the pros and cons of this revival. If there are any, please submit a complaint.

  • GooberBlox

    GooberBlox is a 2016E Roblox revival with an open-source launcher that was made by mathmark824.

    Pros:

    • Finished client
    • Actively documented development
    • Large player base
    • Custom webpage

    Cons:

    • I haven't seen any cons of this revival as of now. If there is any, please send a complaint.

  • Project X Reborn

    Project X is a 2016-2018 revival that allows you to play various old clients. Project X is currently available to play on Windows, iOS, Android and MacOS.

    Pros:

    • Active development
    • Active community

    Cons:

    • Small Community

  • Virtubrick

    Virtubrick is an upcoming 2016 roblox revival that is actively work in progress.

    Pros:

    • Well documented development
    • Custom webpage

    Cons:

    • not playable :(

  • idk16

    IDK16 is an old Roblox revival, trying to revive the era of late 2016. idk16 is trying to use original Roblox software for playing from this period of time, ensuring there is the best accuracy with the experience. idk16 currently support Windows, Android, Linux with Wine and iOS.

    Pros:

    • Supports android, iOS, linux (wine)
    • Owner does testing events sometimes

    Cons:

    • In testing
    • Community isn't that active

  • Atlas (aka Project: Reblock)

    Atlas seems to be an upcoming 2015 Roblox Revival. We currently do not know anything about this revival as it is being worked on.
